# CipherChat

CipherChat is a secure desktop messaging application built with Java.  
It enables encrypted communication between users and stores messages securely in a relational database.
<img width="1536" height="346" alt="cyberchatlogo" src="https://github.com/user-attachments/assets/e7908659-c8c3-423c-bbb2-ca8d6dc04011" />


## Overview

The application implements end-to-end message encryption to ensure that sensitive data remains protected. Passwords are securely hashed and never stored in plain text.

## Tech Stack

- Java  
- Java Swing  
- MySQL  
- JDBC  
- AES encryption  

## Features

- User authentication with secure password hashing  
- Encrypted message storage  
- Conversation-based messaging system  
- Structured layered architecture  

## Purpose

This project was developed to practice secure application design, database modeling, and clean architecture principles in a desktop environment.
